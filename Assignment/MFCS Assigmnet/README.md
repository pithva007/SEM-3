# MFCS Assignment

This folder contains C++ programs for file encoding and decoding, likely part of an assignment for Mathematical Foundations of Computer Science (MFCS).

## Structure
- **`file.cpp`**: Main entry or helper C++ program.
- **`main/`**: Contains encoding and decoding source files (`encode.cpp`, `decode.cpp`), along with test text files like `a.txt` and `Introduction to Algorithms_original.txt`.
- **`final/`**: Contains the final versions of the encoding and decoding programs (`encode.cpp`, `decode.cpp`).

## Usage
Compile the C++ files using a standard C++ compiler (like `g++`).
```bash
g++ encode.cpp -o encode
./encode
```
*(Adjust the compilation commands depending on how the files interdepend).*
