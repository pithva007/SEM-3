# Digital Electronics (DE) Assignment

This directory contains Logisim circuit files (`.circ`) for the Digital Electronics assignment.

## Contents
- **`Assignment_final.circ`**: The final completed circuit design for the assignment.
- **`aa1.circ`**: Additional/helper Logisim circuit file.

## Usage
To open and run these files, you need to have [Logisim](http://www.cburch.com/logisim/) installed. Open Logisim, and use `File > Open` to load the `.circ` files.
