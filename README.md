# SEM-3
Here is all the study material for SEM-3. It includes notes, assignments, and projects for all the subjects in SEM-3. The subjects are:
1. Data Structures and Algorithms
2. Digital Electronics
3. Object Oriented Programming
4. Economics
5. ICC
6. MFCS
7. Assignment of respective subjects
All the materials are organized in folders for each subject. You can find the notes, assignments, and projects in their respective folders. This repository is intended to help students of SEM-3 to easily access and study the materials for their subjects.
Feel free to contribute to this repository by adding more study materials or improving the existing ones. Happy studying!
